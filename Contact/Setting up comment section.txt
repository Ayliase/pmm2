Backend: Handling Submission (PHP)Step 3: Write the Server-Side Script (save_comment.php)This PHP script handles the data flow, security, and saving the data to the database.Connect to the Database: Establish a connection to your MySQL database using credentials (hostname, username, password, database name).Receive and Sanitize Data: Collect the form data using the $_POST superglobal array ($_POST['guest_name'], $_POST['comment_text'], etc.).CRITICAL: Use functions like mysqli_real_escape_string() or prepared statements to sanitize the input. This prevents hackers from inserting malicious code (SQL Injection).Insert Data: Write an SQL INSERT query to add the sanitized data into the comments table.Redirect: After a successful save, redirect the user back to the original blog post URL.PHP<?php
// save_comment.php (Example Logic)

// 1. Database Connection (Replace with your actual details)
$conn = new mysqli("localhost", "db_user", "password", "blog_db");
if ($conn->connect_error) {
    die("Connection failed: " . $conn->connect_error);
}

// 2. Receive and Sanitize Data
$post_id = intval($_POST['post_id']);
$name = mysqli_real_escape_string($conn, $_POST['guest_name']);
$comment = mysqli_real_escape_string($conn, $_POST['comment_text']);

// 3. Insert Data using a Prepared Statement (Safer than direct concatenation)
$stmt = $conn->prepare("INSERT INTO comments (post_id, guest_name, comment_text) VALUES (?, ?, ?)");
$stmt->bind_param("iss", $post_id, $name, $comment); // i=integer, s=string, s=string

if ($stmt->execute()) {
    // 4. Redirect upon success
    header("Location: /blog/post_id=" . $post_id . "#comment-success");
    exit();
} else {
    echo "Error: " . $stmt->error;
}

$stmt->close();
$conn->close();
?>
